Каждый охотник желает знать...

  
  Флейта-фифочка, фланируя фиолетово,
  фуэте, франтишечка, фью-фью-флажолетила.
  
  Свиристит свирель, сосульками стынут странники,
  снег смахнёт симпатия, створя ставенки.
  
  Голосов глагол, говорун-гобой гладь глубинную
  греховодницы гладит, гад, голубиново.
  
  Знам, запотчует зазнобушка земляничная -
  зеленЫм запьём забавушку закадычную.
  
  Жига-жизнь журчит, жмурясь:"Женишься?"
  Жечь жар-птицею - житие женское.
  
  Охрустясь огурчиком, орхеструем оргию -
  обожать отрадушку, охинею окая.
  
  Ку-ку, куколка, купцом куплена, кармы каверза,
  карамелька клитора, камельковая капризА -
   каждый охотник желает знать, где сидит фазан ...
